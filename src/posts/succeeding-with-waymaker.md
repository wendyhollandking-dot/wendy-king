# Succeeding With Waymaker

This capstone project involved redesigning Lumen Learning's introductory module that students complete before starting course content. The revamp — titled "Succeeding With Waymaker" — followed the ADDIE model to create a module that better prepares students for success in Lumen courseware.

## Project Components

The redesign included:

- Identifying learning outcomes
- Writing instructional content
- Creating video content
- Developing slidedecks
- Authoring assessments

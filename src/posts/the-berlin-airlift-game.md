# The Berlin Airlift Game

In my Designing Learning Systems with Simulation and Game Technology course, I created an educational game for use within the Cold War unit of a U.S. History curriculum.

This project represents a practical application of game design principles to historical instruction, demonstrating how interactive simulations can engage students with 20th-century geopolitical content.
